## What is Lockout Sharing?  
[Lockout Sharing](https://wowpedia.fandom.com/wiki/Raid_lockout) in **World of Warcraft** allows multiple characters or players to benefit from the same raid lockout.  
A player with a saved lockout can invite others to their group.  

### üîπ Example:  
If the 9th boss drops a mount, I invite you to a lockout where the first 8 bosses are already killed‚Äîyou join and directly kill the 9th boss.  
‚úÖ Group Lead doesn't need to be present.

---

# üìù How to Use

## üöÄ Getting a lockout

1. Make sure you're **not in a party** and standing **outside the instance** you want to join.
2. Whisper `/w Touchmymana-Draenor !id` to receive a lockout.
3. Follow the instructions provided in chat.

### üîç Optional Commands

- **Normal Mode:** `!normal` or `!nhc` (default)  
- **Heroic Mode:** `!heroic` or `!hc`

### üë• After Entering the Instance

- Want to invite more players? Type `++` ‚Äî you‚Äôll have **30 seconds** to list your group. (Raid:Legacy permits crossfaction listings)
- Otherwise type `+` in chat to confirm you're done ‚Äî the character will leave automatically in **15 seconds**.
